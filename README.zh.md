# 一网统飞 - 无人机巡检系统

## 项目简介
一网统飞-无人机巡检系统是低空政务一体化管控平台，以无人机机巢为硬件载体，通过软硬件深度协同，构建集空域、航线、设备、任务、数据、运维于一体的低空巡检全闭环体系，实现多行业、多场景 “一张网” 统管统飞，全面革新传统政务巡检模式。

## 技术特点
- 🔒 **安全合规部署**：边缘端/国产服务器双部署，数据本地化处理，安全合规、响应迅捷
- ⚡ **极速应急响应**：百米级快速响应应急指令，极端天气、夜间环境均可稳定执行巡检任务
- 🧠 **AI智能提效**：自研AI智能分析算法 + 全自动航线规划，单航线覆盖多巡检目标，巡检效率成倍提升
- 🚀 **7×24小时无人值守**：深度适配各类无人机机巢，支持无人化自动起降、换电/充电，全时段无间断运行

## 核心模块

1. **🛣️ 空域管理**：空域资源统筹规划、飞行权限审批流程化、禁飞/限飞区精准划定、空域态势一张图可视化展示。
2. **✈️ 航线管理**：可视化拖拽式航线绘制、批量航线导入导出、智能避障路径规划、巡检任务模板化配置、历史航线复用与优化。
3. **📡 设备管理**：无人机/机巢状态实时监控、设备在线状态可视化展示、远程控制与参数调节、故障自动告警与定位、全生命周期运维日志记录。
4. **🤖 AI智能识别**：多场景算法模型一键加载，目标自动识别、分类、定位、抓拍，识别结果结构化输出，无需人工回看视频，效率提升80%+。
5. **📝 工单管理**：异常事件秒级自动告警、工单自动生成与智能派发、处置过程全程跟踪、全流程留痕可追溯。
6. **👥 人员管理**：账号权限分级管控、角色自定义配置、操作日志全量审计、巡检人员智能排班、作业绩效自动统计。
7. 🚨 **应急指挥**：应急任务快速下发、现场画面实时回传、多方联动协同指挥、历史任务复盘分析。
8. **💰 充值收费管理**：支持飞行时长/航线次数/巡检面积多维度计费模式，在线充值与账单查询、自动统计与对账、收费数据可视化分析。
